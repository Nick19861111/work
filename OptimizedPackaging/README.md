# 优化打包

优化的主要目的是所有的东西都写一个bat文件下可能有一些功能不需要有的功能有需要，写程序就是要每一个功能都分开然后在组装起来，这样灵活性可以随着自己的组合形成需要的功能。只需要建立一个总的起始类就好

## 功能分类

- ccc的构建
- 热更的打包
- 移动到指定的文件夹
- 图片的压缩
